## Beschreibung:
Dieses Projekt dient als Abschluss des Modul-1 für das Bootcamp von Supercode.

Dabei sollten Wir die bis jetzt erlernten Fähigkeiten zu <html> <css> und <js> in einem beispielhaften Projekt umsetzen.

Das Projekt lautet "Nutrionist" und soll eine Oberfläche für eine "Health-app" darstellen.
Dabei geht es inhaltlich um Ernährung und das Einnehmen von personalisierten Nährstoffen und lädt zum Kauf von diversen Plänen ein.
Umgesetzt wurde hierbei die Main-page.
